# 🤖 AI-3016 — Develop Generative AI Apps in Azure AI Foundry
 
> **Plataforma principal:** [Azure AI Foundry](https://ai.azure.com)

---

## 📌 Descripción del Curso

Este curso es el más directamente centrado en **Azure AI Foundry**.  
Aprenderás a construir aplicaciones de IA generativa que utilizan modelos de lenguaje, prompt flow y RAG para aportar valor real a los usuarios.

La IA generativa es cada vez más accesible gracias a plataformas como Azure AI Foundry.  
Este curso te prepara para la credencial **Microsoft Applied Skills: APL-3016**.

## 🧠 Temas Principales

### Módulo 1 — Introducción a Azure AI Foundry
- ¿Qué es la IA generativa?
- Descripción de Azure AI Foundry y sus capacidades
- Herramientas para desarrolladores y SDKs disponibles
- Principios de IA Responsable

### Módulo 2 — Crear un AI Project
- Creación de un AI Hub en Azure
- Configuración de un nuevo Azure AI Project
- Asignación de roles y usuarios al proyecto
- Configuración de recursos conectados (AI Services, AI Search, Storage)

### Módulo 3 — Model Catalog
- Exploración del catálogo de modelos en Azure AI Foundry
- Selección del modelo adecuado según el caso de uso
- Configuración de parámetros de despliegue

### Módulo 4 — Playground y Prompt Engineering
- Uso del Playground para pruebas interactivas
- Técnicas de prompt engineering
- Optimización del rendimiento del modelo

### Módulo 5 — Deploy de Modelos
- Despliegue de modelos a un endpoint
- Edición y actualización de modelos desplegados
- Pruebas post-despliegue

### Módulo 6 — Prompt Flow
- Ciclo de vida del desarrollo de apps con LLMs
- Componentes principales: nodos de modelo, Python y prompt
- Definición de entradas/salidas y ejecución del flujo

### Módulo 7 — RAG (Retrieval-Augmented Generation)
- Creación de componentes de datos e índices con Azure AI Search
- Integración del índice en el flujo de chat
- Construcción de un agente con datos propios

### Módulo 8 — Fine-Tuning y Evaluación
- Cuándo y cómo hacer fine-tuning de un modelo
- Evaluación del rendimiento con métricas de calidad y seguridad
- Evaluaciones manuales y asistidas por IA

### Módulo 9 — IA Responsable
- Filtros de contenido personalizados (input/output)
- Blocklists y material protegido
- Implementación responsable en producción

## 🎯 Objetivos de Aprendizaje

Al finalizar este curso, el participante será capaz de:

- ✅ Describir los servicios de Azure AI y cuándo usarlos
- ✅ Crear y configurar un proyecto en Azure AI Foundry
- ✅ Seleccionar y desplegar modelos desde el catálogo de modelos
- ✅ Desarrollar aplicaciones de IA usando el Azure AI Foundry SDK
- ✅ Construir soluciones RAG con datos propios
- ✅ Hacer fine-tuning de modelos de lenguaje
- ✅ Evaluar el rendimiento de apps de IA generativa
- ✅ Implementar controles de IA Responsable
